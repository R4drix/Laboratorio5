2. Redes y protocolos básicos

Las redes conectan dispositivos para compartir datos. Para que esa comunicación funcione existen protocolos —conjuntos de reglas— que definen cómo se empaquetan, envían y reciben los datos. Ejemplos conocidos son TCP/IP para transmisión fiable y HTTP/HTTPS para la web. Conocer el modelo en capas (por ejemplo, el modelo OSI) ayuda a entender dónde actúa cada protocolo y cómo se puede diagnosticar un problema de conectividad.
